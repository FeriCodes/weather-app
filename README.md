# 🌤️ Simple Weather Web

This is a simple weather website for my neighborhood. It also has a search bar so other users can check the weather for their cities or coordinates.

## 🎯 Features

- **Search Bar:** You can search by city name or exact location.
- **Home Button:** A button to quickly show the weather for my neighborhood.
- **Weather Information:** It shows Temperature, Feels Like, Humidity, and Wind Speed.

## 🧠 What I Learned

In this project, I learned many good things:
- How to connect to an API and read **JSON** files.
- How to make a web application using **Flask**.
- How to send data from Python to a web page with `render_template`.
- How to build the page structure with **HTML**.
- How to make the website look beautiful and modern with **CSS**.
